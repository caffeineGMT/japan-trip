## Emergency Procedures

### If Site Goes Down

**Immediate Actions:**
1. Post on ProductHunt: "We're experiencing higher than expected traffic. Team is on it. Back online in [X] minutes."
2. Tweet: "Site is temporarily down due to high traffic 🚀😅 Working on it!"
3. Check Vercel logs for error
4. Scale serverless functions if needed
5. Enable maintenance mode if necessary
6. Update every 15 minutes until resolved

**Template:**
```
Update: Site is back online!

Sorry for the brief downtime. We underestimated how many amazing people would
check this out today 🙏

Everything's working now. Thanks for your patience!
```

### If Payments Fail

**Immediate Actions:**
1. Disable checkout temporarily (prevent more failures)
2. Contact Stripe support immediately
3. Communicate transparently on PH
4. Offer alternative payment method (PayPal)
5. Manually process any failed orders
6. Follow up with affected customers personally

**Template:**
```
We're experiencing a technical issue with our payment processor. If you tried
to sign up and it failed, please DM me directly - I'll make sure you get access
and your early bird pricing honored.

Sorry for the inconvenience! 🙏
```

### If Hunter Disappears

**Backup Plan:**
1. Wait 30 minutes
2. Text/call hunter if you have their number
3. If no response, post yourself
4. Explain briefly: "Hunter had emergency, launching solo"
5. Don't make it dramatic
6. Move on with the plan

### If Rank Isn't Good

**Reality Check:**
- Not every launch goes viral
- Engagement > rank
- Learning > perfection
- Relationships built > upvotes
- It's a marathon, not a sprint

**What to Do:**
- Stay positive and engaged
- Focus on quality conversations
- Build genuine connections
- Use it as learning experience
- There's always next launch

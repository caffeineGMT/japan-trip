Generate native speaker audio for Japanese phrases

📝 Instructions

  • Select a Japanese voice from your browser's available voices
  • Click "Generate All Audio" to create MP3 files for all phrases
  • The audio files will be downloaded automatically as a ZIP file
  • Extract the ZIP and place the files in the /audio/phrases/ directory
  • For best quality, use a browser with high-quality Japanese voices (Chrome recommended)

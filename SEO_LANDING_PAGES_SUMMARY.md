## Key Decisions

1. **Fallback Content:** Used high-quality SEO-optimized descriptions instead of relying on Wikipedia API (more reliable, better optimized)
2. **Unsplash Source:** Used free image service as fallback (no API key required for MVP)
3. **Priority Scoring:** Weighted by search volume (Tokyo/Kyoto/Osaka = 0.9, others 0.7-0.8)
4. **Internal Linking:** Random 5 related cities per page (better crawlability)
5. **Mobile-First:** Responsive design for 60%+ mobile travel searches

- [ ] Review tomorrow's content from 30-day plan

- [ ] Edit Reel in CapCut/InShot (15-30 min)

- [ ] Export in 9:16 ratio, 1080x1920px

- [ ] Monitor first hour performance
- [ ] Respond to comments within 1 hour
- [ ] Share to Stories with "New Reel" sticker

- [ ] Track metrics in Performance Log

## Editing Workflow (20-Minute Template)

**Minute 0-5: Setup**
1. Import footage into CapCut
2. Set project to 9:16 ratio
3. Trim clips to 3-5 second segments

**Minute 5-10: Sequencing**
4. Arrange clips in story order
5. Add transitions (quick cuts work best)
6. Trim to 15-25 seconds total

**Minute 10-15: Enhancement**
7. Add trending audio (align with beat drops)
8. Color correct (boost saturation +10-15)
9. Add subtle zoom/pan effects

**Minute 15-20: Text & Export**
10. Add hook text overlay (0-3s, large font)
11. Add CTA text overlay (final 2s)
12. Export 1080x1920, 30fps
13. Preview on phone before posting

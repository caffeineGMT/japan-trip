## 🎬 Content Production Tips

### Hook Formula (First 3 Seconds)
1. **Pattern Interrupt:** "Everyone does X... but THIS..."
2. **Shocking Statement:** "I wasted $300 because..."
3. **Curiosity Gap:** "This temple is [surprising fact]"
4. **Time-Sensitive:** "🌸 LIVE UPDATE:"
5. **Relatable Pain:** "Lost in Tokyo with no WiFi?"

### 20-Minute Editing Workflow
1. (0-5 min) Import footage, set 9:16 ratio, trim clips
2. (5-10 min) Sequence clips, add transitions
3. (10-15 min) Add trending audio, color correct
4. (15-20 min) Add text overlays (hook + CTA), export

### Stock Footage Search Terms
- "Tokyo street timelapse"
- "Kyoto temple peaceful"
- "Cherry blossom sakura"
- "Japanese bamboo forest"
- "Mt Fuji sunrise"
- "Osaka street food"

## 🔧 Troubleshooting Guide

| Problem | Likely Cause | Solution |
|---------|--------------|----------|
| Low views (<500) | Weak hook, wrong audio | Test different hooks, use trending audio |
| Views but no profile visits | Unclear CTA | Add stronger text overlay CTA |
| Profile visits but no clicks | Bio doesn't match promise | Optimize bio, simplify link |
| Clicks but no signups | Landing page doesn't convert | Match headline to Reel, add social proof |
| Signups but no paid | Free tier too generous | Limit free features, add urgency |

# Instagram Reels Campaign Implementation Guide

## Executive Summary

**Campaign:** "Hidden Japan Spots" - 30-day Instagram Reels series
**Target:** #japantravel (14M posts) audience
**Goal:** 50K views → 1,000 profile visits → 200 clicks → 20 signups → 2 paid ($20 MRR)

## Content Production Workflow

### Batch Production Strategy (Recommended)

**Option 1: Weekly Batch (Sundays, 3 hours)**
- Produce all 7 Reels for upcoming week
- Download all stock footage needed
- Edit all videos in one session
- Schedule posts for week

**Benefits:**
- More efficient (get into flow state)
- Consistent quality
- Less daily stress
- Can optimize cross-Reel themes

**Option 2: Daily Production (30 min/day)**
- Produce today's Reel each morning
- More reactive to trends
- Fresh content daily

**Benefits:**
- Can incorporate trending audio faster
- Respond to community feedback
- More flexible

**Recommended:** Start with daily, move to batch after Week 1

## Troubleshooting Common Issues

### Problem: Low Views (<500 per Reel)

**Diagnoses:**
- Weak hook (first 3 seconds didn't grab attention)
- Using non-trending audio
- Posting at wrong time
- Content not relevant to #japantravel audience

**Fixes:**
- Test different hook styles (use template variations)
- Check Instagram trending audio daily
- Post at 7am, 12pm, or 7pm PT
- Make content more Japan-specific

---

### Problem: Views But No Profile Visits

**Diagnoses:**
- CTA not clear or compelling
- Bio doesn't match Reel content
- No reason to click profile

**Fixes:**
- Add stronger CTA overlay on final frame
- Test "Link in bio" vs "Try free - bio" vs "Full guide in bio"
- Optimize bio to match Reel promise
- Add "tap here for more hidden spots" prompt

---

### Problem: Profile Visits But No Link Clicks

**Diagnoses:**
- Bio link unclear
- Bio doesn't convey value
- Too many distractions in bio

**Fixes:**
- Simplify bio to 1 clear CTA
- Make link benefit obvious ("Free AI planner" not "Learn more")
- Test different bio versions weekly
- Add urgency ("Limited free spots")

---

### Problem: Link Clicks But No Signups

**Diagnoses:**
- Landing page doesn't convert
- Too much friction in signup
- Value prop unclear

**Fixes:**
- Match landing page headline to Reel hook
- Simplify signup form (email only)
- Add social proof (testimonials, user count)
- Show app screenshots above fold

---

### Problem: Signups But No Paid Conversions

**Diagnoses:**
- Free tier too generous
- No urgency to upgrade
- Pricing unclear

**Fixes:**
- Limit free features (e.g., only 3 days of itinerary on free)
- Add upgrade prompts at key moments
- Test limited-time offers
- Show value of Premium (before/after comparison)

- Use emergency content templates (5-minute Reels)
- Screenshot carousels instead of video
- Repost user-generated content
- Share stats/facts with text overlays only

**Option 2: Reduce Frequency**
- Switch to 5 Reels/week instead of 7
- Quality > quantity
- Focus on best-performing content types only

**Option 3: Outsource**

- Buy pre-made Japan travel B-roll ($50/month)
- Partner with more influencers to create content

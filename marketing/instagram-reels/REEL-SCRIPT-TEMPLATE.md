# Instagram Reel Script Template

## How to Use This Template

1. **Copy the template** for your specific content type
2. **Fill in the brackets** with your specific content
3. **Keep it under 30 seconds** (15-20 is ideal)
4. **Hook in first 3 seconds** (pattern interrupt, shocking stat, curiosity)
5. **CTA at the end** (simple, clear, actionable)

## Quick Production Checklist

### Before You Start
- [ ] Content type selected from 30-day plan
- [ ] Script filled out using template above
- [ ] Stock footage downloaded OR screen recording ready
- [ ] Trending audio identified (check Instagram trending)
- [ ] Hashtags selected (8-10 from strategy)

### During Editing (CapCut/InShot)
- [ ] Aspect ratio set to 9:16 (1080x1920)
- [ ] Clips trimmed to 3-5 seconds each
- [ ] Total length: 15-25 seconds
- [ ] Hook text overlay (0-3s, large font, high contrast)
- [ ] Supporting text overlays (clear, readable on mobile)
- [ ] CTA text overlay (final 2-3s)
- [ ] Trending audio added and synced
- [ ] Color correction: +10-15 saturation boost
- [ ] Preview on phone before export

### After Export
- [ ] Upload to Instagram Reels
- [ ] Add caption (1-2 sentences, conversational)
- [ ] Add hashtags (in caption or first comment)
- [ ] Add location tag (if relevant: Tokyo, Kyoto, etc.)
- [ ] Share to Stories immediately
- [ ] Set reminder to respond to comments in 1 hour

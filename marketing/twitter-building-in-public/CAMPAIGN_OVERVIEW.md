# Twitter 'Building in Public' Campaign
## Japan Trip Companion Web App

---

## Campaign Goals

**Primary Objective**: Build audience and authority on Twitter by transparently sharing the journey of building a Japan trip planning web app.

**Target Metrics (Month 3)**:
- 500 followers
- 50 clicks/week to landing page
- 5 signups/month from Twitter
- $5 MRR (0.5 paid conversions)
- 100+ engaged community members

**Long-term Goals**:
- Establish authority in travel tech space
- Generate press coverage opportunities
- Create partnership opportunities with travel brands
- Build email list for product launches

   - Looking for white-label solutions

## Content Pillars

### 1. Daily Shipping Updates
Format: "[Day X] Shipped: [Feature]. [One-line impact]. [Screenshot/demo]"
Example: "Day 7: Shipped offline maps with 500+ tiles cached. Users can now plan in the subway. Here's the service worker code: [screenshot]"

### 2. Metrics Transparency
Format: "Week X numbers: [Metric 1], [Metric 2], [Metric 3]. [Key insight]."
Example: "Week 2 numbers: 87 users (+45), $149 revenue (+$99), 12% conversion. Learning: Cherry blossom feature drove 80% of signups."

### 3. Problem-Solving
Format: "[Problem] → [Solution]. [What I learned]."
Example: "Cherry blossom API went down → Built fallback scraper in 2 hours. Lesson: Always have a Plan B for critical features."

### 4. User Stories
Format: "User story: [Quote from user]. [How we helped]. [CTA]."
Example: "User: 'I planned my entire 2-week trip in 15 minutes!' This is why we build. Want to try? [link]"

### 5. Learning in Public
Format: "TIL: [Technical insight]. [How I applied it]. [Code snippet]."
Example: "TIL: Service workers cache strategies. Implemented network-first for API calls, cache-first for maps. 90% faster repeat loads."

## Risk Mitigation

**Avoid These Mistakes**:
- Inconsistent posting (kills momentum)
- Ignoring comments (kills community)
- Too much promotion (kills trust)
- Comparing to others (kills motivation)
- Quitting too early (most quit before results)

**When to Pivot**:
- If engagement drops 50% for 2 weeks straight
- If follower growth stalls completely
- If content feels forced/inauthentic
- If Twitter isn't driving any conversions after 60 days

**Backup Plan**:
- Document learnings for blog posts
- Repurpose Twitter content for LinkedIn
- Create YouTube videos from popular threads
- Compile into email newsletter
